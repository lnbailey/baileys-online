# Bailey Family Album

A photo album. Database to hold family photos, searchable by Name, Location, Event, etc...

## MVP:
* Ability to store photos in Firebase
* Ability to upload photos with details
* Ability to search and display photos in a gallery

## Stretch Goals:
* Ability to restrict photo upload to only authorized users
* Custom theme for galleries (set as default by user or choose after search)
* Ability to export slide show
